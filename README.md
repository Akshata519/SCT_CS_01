# SCT_CS_01
# Caesar Cipher Encryption & Decryption

## Project Overview

This project is a simple implementation of the Caesar Cipher algorithm developed as part of a Cyber Security Internship. The application allows users to encrypt and decrypt text messages using a user-defined shift value.

## Features

- Encrypt text using Caesar Cipher
- Decrypt encrypted text
- User-defined shift value
- Simple menu-driven interface
- Preserves uppercase, lowercase, spaces, and symbols

## Technologies Used

- Python 3

## How to Run

1. Install Python 3.
2. Save the program as `caesar_cipher.py`.
3. Open a terminal or Python IDLE.
4. Run the program:
   ```
   python caesar_cipher.py
   ```
5. Follow the on-screen instructions.

## Example

Message: Hello World

Shift: 3

Encrypted: Khoor Zruog

Decrypted: Hello World
